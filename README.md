# TLN_AUM
Triple La Niña effect on the Australian Monsoon

To read this code you will need access to NCI and the following project: 

oi10 : For CMIP6 data

rt52 : For ERA5 data

zv2 : For AGCD data

iu04 : For AWRA-L data


And download the following data:
ERSSTV5 : via https://psl.noaa.gov/data/gridded/data.noaa.ersst.v5.html
